# 🛡️ Travel Safe 프로젝트 스킬 로드맵

이 파일은 **Travel Safe** 앱을 완성하기 위해 필요한 기술들과 각 기술의 목적을 정리한 가이드입니다. 하나씩 마스터하며 앱을 완성해 보세요!

---

## 🛠️ Level 1: 프로젝트 기초 (Foundation)
프로젝트를 관리하고 시작하는 가장 기본이 되는 기술입니다.

- [x] **Git & GitHub**: 코드의 변경 이력을 관리하고 온라인에 안전하게 저장하는 기술.
- [ ] **Vite**: 최신 웹 개발 환경을 빠르고 가볍게 구축해 주는 도구.
- [ ] **npm (Node Package Manager)**: 필요한 도구(라이브러리)들을 설치하고 관리하는 도구.

---

## 🎨 Level 2: 사용자 경험 & 디자인 (UI/UX)
사용자가 보게 될 "프리미엄"한 화면을 만드는 기술입니다.

- [ ] **React**: 화면을 '컴포넌트'라는 작은 단위로 쪼개서 스마트하게 관리하는 자바스크립트 라이브러리.
- [ ] **Modern CSS (Vanilla)**:
    - **Flexbox/Grid**: 화면의 레이아웃을 자유자재로 배치하는 기술.
    - **Glassmorphism**: 반투명한 유리 느낌의 고급스러운 효과를 내는 기술.
    - **Keyframe Animation**: 부드러운 버튼 클릭감이나 화면 전환 효과를 만드는 기술.

---

## 📍 Level 3: 지도 & 위치 정보 (Maps & Geo)
이 앱의 핵심인 '위치 추적'과 '지도 표시'를 위한 기술입니다.

- [ ] **Google Maps JavaScript API**: 지도 위에 마커를 찍고, 선(Polyline)을 긋는 기능을 구현합니다.
- [ ] **HTML5 Geolocation API**: 사용자의 스마트폰에서 현재 위치(위도, 경도)를 알아내는 내장 기능입니다.

---

## 🧠 Level 4: 데이터와 동기화 (Backend)
가족끼리 정보를 실시간으로 주고받게 해주는 앱의 '두뇌' 역할입니다.

- [ ] **Firebase Authentication**: 안전하게 로그인하고 회원가입을 처리하는 기능.
- [ ] **Cloud Firestore (NoSQL)**: 실시간으로 위치 정보를 가족들에게 쏴주는 데이터베이스 기술.
- [ ] **Firebase Storage**: 여행지에서 찍은 고화질 사진을 클라우드에 저장하는 기술.

---

## 🔔 Level 5: 고급 기능 (Advanced)
앱을 더욱 가치 있게 만드는 특별한 기능들입니다.

- [ ] **PWA (Progressive Web App)**: 웹사이트를 실제 앱처럼 스마트폰 홈 화면에 설치하고 푸시 알림을 보낼 수 있게 만드는 기술.
- [ ] **Real-time SOS Alert**: 데이터베이스의 변화를 감지해 가족들에게 즉시 위급 상황을 알리는 로직.

---

> **💡 TIP**: 너무 한꺼번에 다 배우려 하지 마세요! 제가 필요한 단계마다 하나씩 상세히 설명해 드리면서 함께 만들어 갈 것입니다. 한 번에 하나씩, **Level 1**부터 천천히 격파해 봅시다! 🚀
